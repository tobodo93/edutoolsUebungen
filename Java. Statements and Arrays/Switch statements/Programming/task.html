Write a program that reads the number of a shape (1 – square, 2 – circle, 3 – triangle, 4 – rhombus) and prints You have chosen a square (or circle, or triangle, or rhombus, depending on the number). If the number does not correspond to any of the listed shapes, the program should output, There is no such shape!.

Note: Please make sure your output exactly matches the required string, including capitals, spaces, and punctuation marks.


Sample Input:
1
Sample Output:
You have chosen a square


Memory limit: 256 Mb
Time limit: 5s
